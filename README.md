# Security and Robustness of Spatial Anonymization Techniques

This repository contains the scripts and experimental results used in the paper:

"Segurança e Robustez de Estratégias para Anonimização e De-Anonimização de Dados Sensíveis de Localização"

## Contents
- `scripts/`: implementation of anonymization techniques and de-anonymization attacks
- `notebooks/`: Jupyter notebooks used for experiments and figure generation
- `results/`: aggregated experimental results and figures

## Dataset
The experiments use public georeferenced data from the project "Onde Está o Aedes?"
(Porto Alegre municipality). The raw dataset is **not redistributed** in this repository.


## Reproducibility
All experiments were conducted using Python 3.10.
